## 核心教训

### 1. 复杂 = 死亡

本轮回测验证了一个强烈规律：**条件越多的策略表现越差**。

```
多因子评分(S79)    ← 3因子等权评分 + TOP40 → 失败
趋势强度四确认(S80) ← 4个AND条件叠加 → 失败  
波动率分层动量(S82) ← 分三层+动量前15% → 成功(唯一成功复杂策略)
双频段动量(S81)    ← 2个动量条件相加 → 成功
```

成功的复杂策略(S82)使用了**分层+子集内选优**而非**AND叠加**——这是关键区别。

### 2. 低价策略仍然统治5年周期

```
S25 低价日频    25.15%  ← 纯低价
S66 低价+排除  25.09%  ← 低价+简单过滤
S67 低价周频   24.97%  ← 低价+改频率
S78 低价月频   23.75%  ← 低价+降频
S87 低价月频+排 23.49%  ← 低价+过滤+降频
```

2021-2025这5年A股小盘行情下，低价因子压倒一切。

### 3. 频率递减规律

```
日频  25.15% 换手6.68%  成本≈20%
周频  24.97% 换手3.87%  成本≈12%
月频  23.75% 换手2.53%  成本≈8%
```

收益每步降~1%，但换手每步降~40%。月频是性价比之选。

### 4. 唯一成功的非低价策略

- S57 超涨反向：18.75%, 夏普1.06 — 宽基增强
- S58 布林上轨反向：16.66%, 夏普1.12 — 宽基增强
- S82 波动率分层动量：16.28%, 夏普0.45 — 分层选优

非低价策略能做到15-18%年化，但需要独特逻辑。
