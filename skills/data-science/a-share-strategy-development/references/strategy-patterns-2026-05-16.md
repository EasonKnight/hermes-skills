## 关键教训

### 1. 「选股太少」是最常见的策略致死原因

S79和S83的共同问题：多重条件叠加后，每天买不到几只股票。

**验证方法：**
```python
per_day = signal.sum(axis=0)
print(f"日均持股: {per_day.mean():.0f} 只, 信号天数: {(per_day>0).sum()}")
```

**修复方向：**
- 放宽某个条件的分位数（如从30%→50%）
- 减少条件数量（3个条件→2个）
- 用「或」代替「与」（扩大候选池）
- 改为固定数量选股（top-N）而非阈值过滤

### 2. 分层策略有效但需大盘支撑

S82 波动率分层：把股票按波动率低/中/高均分三组，只从低中两层选动量股。这比 S76 直接低波动过滤更精细，但需要足够多的候选股（低+中层加起来占2/3市场）。

**分层 vs 简单过滤：**
```python
# 简单过滤（S76风格）
candidates = low_vol  # 只选低波动50%

# 分层策略（S82风格）
candidates = low_vol | mid_vol  # 选低+中波动66%
```

### 3. 多因子评分需要平衡因子权重

S79 三因子等权评分（动量+低波+低价），选总分前40。问题：三个因子中低价因子占据了主导（A股低价股很多），导致动量因子几乎不起作用。

**改进方向：** 给每个因子不同的权重，或者先按一个因子筛选再在子集中按另一因子排序（顺序过滤而非加权）。

### 4. 新增策略全在周频

5个新策略全部采用周频。这不是巧合——从经验看，日频策略的换手率在A股市场环境中普遍过高（50-87%换手），成本吃掉大部分alpha。
