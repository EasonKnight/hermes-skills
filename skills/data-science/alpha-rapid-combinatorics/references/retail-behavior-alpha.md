# 散户行为逆转因子（Retail Flow Reversal Alpha）

## 理论基础

散户亏钱有五个可量化的行为金融学规律：

| 行为 | 代理 Proxy | 预期逆向方向 |
|------|-----------|------------|
| 追涨（FOMO） | 短期大涨 + 放量 | 做空（反转跌） |
| 恐慌割肉 | 短期暴跌 + 放量 | 做多（超跌反弹） |
| 低价偏好（彩票效应） | 绝对低价 | 做空（质量差） |
| 换手率过高（过度交易） | 成交量/成交额异动 | 做空（摩擦成本） |
| 注意力驱动（追涨停） | 大涨后换手率激增 | 做空（高位接盘） |

## 因子设计模式（全矩阵 batch）

### 基础核：双侧动量反转
```python
ret5 = ret_n_batch(close, 5)
amt_r = amount_ratio_batch(close, volume, 20)
h = -ret5 * amt_r
```
- `ret_5d > 0`（涨） → `-ret5 < 0` → 做空（追涨反转）
- `ret_5d < 0`（跌） → `-ret5 > 0` → 做多（恐慌反弹）
- 放量（amt_r > 1）放大信号强度

### 变体1：低位恐慌反转
```python
pp60 = price_position_batch(close, 60)
h = -ret5 * amt_r * (1 - pp60)
```
- 加权价格近60日低位的股票（`pp60 → 0`）
- 捕捉"散户在地板割肉"后的资金回流

### 变体2：高位追涨反转
```python
ret20 = ret_n_batch(close, 20)
amt_r5 = amount_ratio_batch(close, volume, 5)
pp60 = price_position_batch(close, 60)
h = -ret20 * amt_r5 * pp60
```
- 20日趋势确认 + 5日暴量确认 + 高位确认（三重过滤）
- 捕捉"散户高位追涨停"后的反转

## 需要的输入字段
- `close` — 收盘价（必须）
- `volume` — 成交量（必须）
- 不需要 Level-2 逐笔数据，用**量价异动**近似散户行为

## 注意事项
1. **必须取负号**：散户追涨 = 我们的做空信号
2. **放量是关键确认**：单纯短期涨跌可能只是噪声，+放量才确认散户参与
3. **与 Amihud 非流动性正交**：不放量策略（非流动性）完全是另一方向，可并行探索
4. **价格位置作为条件增强**：高位pp60>0.7 追涨反转更可靠，低位pp60<0.3恐慌反弹更可靠
