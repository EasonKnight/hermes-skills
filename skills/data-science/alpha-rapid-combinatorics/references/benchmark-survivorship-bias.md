# 中证1000基准幸存者偏差修复（纳入日期法）

## 问题

`_compute_benchmark` 使用当前的 CSI1000 成分股（775只）回溯整个10年。幸存者偏差使基准收益虚高：
- 跌出成分股的股票（表现差）被排除
- 后来涨大进入成分股（表现好）被包含
- 早期实际成分股数 ~1000，但当前名单中2016年只有~123只能

## 修复方法

使用 `index_stock_cons(symbol="000852")` 返回的 `纳入日期` 字段，对每只股票只在其进入指数之后才计入基准：

```python
def get_csi1000_at_date(df, date):
    codes = df[df["纳入日期"] <= pd.Timestamp(date)]["品种代码"].astype(str).str.zfill(6).tolist()
    return set(codes)
```

## 效果

| 状态 | 基准收益（10年） | 年化 |
|------|:--------------:|:---:|
| 修复前（当前成分股全周期） | +224% | 12.5% |
| 修复后（按纳入日期过滤） | +145% | 9.3% |
| 中证1000指数（市值加权） | +121% | 8.2% |

## 局限

修复后早期成分股数仍偏少（2016年约123只，真实~1000只），因为已剔除的股票无纳入日期记录。未来可用 `index_stock_cons_weight_csindex` 从 csindex 下载历史快照来完全修复。
