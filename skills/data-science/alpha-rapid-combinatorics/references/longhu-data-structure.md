# 龙虎榜数据结构（akshare）

从东方财富获取的龙虎榜数据，可用于构建散户行为因子（营业部散户跟风、游资接力、机构净买等信号）。

## 核心接口一览

| 函数 | 说明 | 频率 |
|------|------|------|
| `stock_lhb_detail_em(start_date, end_date)` | 每日龙虎榜详情（最核心） | 日频 |
| `stock_lhb_stock_statistic_em(symbol)` | 个股上榜统计 | 近一月/三月/半年/一年 |
| `stock_lhb_stock_detail_em(symbol, date, flag)` | 单只个股某日席位明细 | 日频（单日） |
| `stock_lhb_stock_detail_date_em(symbol, date, flag)` | 同上（日期版别名） | 日频（单日） |
| `stock_lhb_jgstatistic_em(symbol)` | 机构龙虎榜统计 | 近一月/三月/半年/一年 |
| `stock_lhb_yybph_em(symbol)` | 营业部排名（游资胜率） | 近一月/三月/半年/一年 |
| `stock_lhb_traderstatistic_em(symbol)` | 交易席位统计 | 近一月/三月/半年/一年 |
| `stock_lhb_jgmmtj_em(symbol)` | 机构买卖统计 | 近一月/三月/半年/一年 |
| `stock_lhb_yyb_detail_em(symbol, start_date, end_date)` | 营业部龙虎榜明细 | 日频 |

## stock_lhb_detail_em — 每日详情（最常用）

```python
import akshare as ak
df = ak.stock_lhb_detail_em(start_date='20260515', end_date='20260518')
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `序号` | int | - |
| `代码` | str | 股票代码 |
| `名称` | str | 股票名称 |
| `上榜日` | date | - |
| `解读` | str | 如"1家机构买入，成功率47.25%" |
| `收盘价` | float | - |
| `涨跌幅` | float | % |
| `龙虎榜净买额` | float | 元（正=主力买入，负=主力卖出）⭐ |
| `龙虎榜买入额` | float | 元 |
| `龙虎榜卖出额` | float | 元 |
| `龙虎榜成交额` | float | 元 |
| `市场总成交额` | float | 该股当天全部成交额 |
| `净买额占总成交比` | float | % ⭐ |
| `成交额占总成交比` | float | % ⭐ |
| `换手率` | float | % |
| `流通市值` | float | 元 |
| `上榜原因` | str | 见下方原因类型 |
| `上榜后1日` | float | 后续涨跌幅% |
| `上榜后2日` | float | 后续涨跌幅% |
| `上榜后5日` | float | 后续涨跌幅% |
| `上榜后10日` | float | 后续涨跌幅% |

## stock_lhb_stock_statistic_em — 个股上榜统计

```python
df = ak.stock_lhb_stock_statistic_em(symbol='近一月')  # 近一月/近三月/近六月/近一年
```

比详情表多出：
- `上榜次数` — 统计期内上榜总次数
- `买方机构次数`, `卖方机构次数` — 机构参与频率
- `机构买入净额`, `机构买入总额`, `机构卖出总额`
- `近1个月涨跌幅`, `近3个月涨跌幅`, `近6个月涨跌幅`, `近1年涨跌幅`

## stock_lhb_jgstatistic_em — 机构龙虎榜统计

```python
df = ak.stock_lhb_jgstatistic_em(symbol='近一月')
```

| 字段 | 意义 |
|------|------|
| `机构买入额/卖出额` | 机构在龙虎榜上的买卖金额 |
| `机构买入次数/卖出次数` | 机构上榜次数 |
| `机构净买额` | 买入额-卖出额，正=机构偏向买入 |

## stock_lhb_yybph_em — 营业部排行（游资胜率）

```python
df = ak.stock_lhb_yybph_em(symbol='近一月')
```

| 字段 | 意义 |
|------|------|
| `营业部名称` | 游资席位名 |
| `上榜后1天-买入次数` |该营业部买入后，股价在 N 天后的表现统计 |
| `上榜后1天-平均涨幅` | 平均涨跌幅% |
| `上榜后1天-上涨概率` | 上涨比例% |
| ...同上到`上榜后10天` | |

可用于构建**游资跟单/反指信号**，如"低胜率营业部买入 = 做空信号"。

## stock_lhb_stock_detail_em — 个股席位明细（买卖双方）

```python
# flag='买入' 查看买方前5，flag='卖出' 查看卖方前5
df = ak.stock_lhb_stock_detail_em(symbol='000021', date='20260518', flag='买入')
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `序号` | int | - |
| `交易营业部名称` | str | 席位/营业部全称 |
| `买入金额` | float | 元 |
| `买入金额-占总成交比例` | float | 占该股当天总成交额比例 |
| `卖出金额` | float | 元 |
| `卖出金额-占总成交比例` | float | - |
| `净额` | float | 元，买入金额-卖出金额 |
| `类型` | str | 对应的上榜原因 |

**典型结果**（深科技 000021 2026-05-18 涨停买方）：

| 营业部 | 买入金额 | 净额 | 识别 |
|---|---|---|---|
| 深股通专用 | 2.87亿 | +2.24亿 | 北向资金 |
| 国泰海通成都北一环 | 2.01亿 | +2.00亿 | 游资 |
| 机构专用 | 8481万 | +7616万 | 机构 |
| 华鑫证券上海陆家嘴 | 8302万 | +8284万 | 游资 |

**用途**：识别每只龙虎榜股票的买卖双方类型（北向/机构/游资/散户跟风营业部）。

## 上榜原因类型（从实际数据抓取）

| 原因关键词 | 含义 |
|-----------|------|
| 日涨幅偏离值达到7% | 当天涨超7%（追涨信号） |
| 日跌幅偏离值达到7% | 当天跌超7%（恐慌信号） |
| 日换手率达到20% | 换手率极高（散户交易活跃） |
| 连续三个交易日内涨跌幅偏离值累计达到20% | 连续异动 |
| ST证券...偏离值达到12% | ST股异动 |
| 无价格涨跌幅限制 | 新股上市首日 |

## 散户行为因子构建方向

### 方向1：龙虎榜净买额反指
- `净买额占总成交比` 极高（>20%）且上榜原因为"涨幅偏离值" → 散户跟风买 → 做空
- `净买额占总成交比` 极低（负大值）且上榜原因为"跌幅偏离值" → 散户恐慌卖 → 做多

### 方向2：机构 vs 游资背离
- 机构净买额 > 0 但 龙虎榜净买额 < 0 → 机构买入但游资/散户卖出 → 看多
- 机构净买额 < 0 但 龙虎榜净买额 > 0 → 机构卖出但游资/散户买入 → 看空

### 方向3：营业部历史胜率筛选
- 低胜率营业部（上榜后1日上涨概率<40%）买入 → 做空信号
- 高胜率营业部（上榜后1日上涨概率>60%）买入 → 做多信号

## 注意

1. 龙虎榜仅收录**有异常波动的股票**（涨停/跌停/换手率异常等），不是全市场覆盖
2. 数据在实际交易日**收盘后**才发布（约17:00-18:00）
3. 所有金额字段均为**元**（不是万元），可直接用于运算
4. `净买额占总成交比` 和 `成交额占总成交比` 已是百分比数值（如 13.2 表示 13.2%）
5. `上榜后N日` 也是百分比值（如 1.73 表示 +1.73%）
