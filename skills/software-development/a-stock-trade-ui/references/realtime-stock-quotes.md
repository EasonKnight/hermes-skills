# A股实时行情 — 新浪免费接口

## 接口地址

```
http://hq.sinajs.cn/list={code1},{code2},...
```

## 代码格式

| 交易所 | 前缀 | 示例 |
|--------|------|------|
| 沪市 A 股 | `sh` | sh600519 |
| 深市 A 股 | `sz` | sz000001 |

前缀映射：
```python
prefixes = {"000":"sz","001":"sz","002":"sz","003":"sz",
            "300":"sz","301":"sz","302":"sz","303":"sz",
            "600":"sh","601":"sh","603":"sh","605":"sh","688":"sh"}
```

## 请求头

必须带 `Referer` 头，否则可能被拦截：
```python
req = urllib.request.Request(url,
    headers={"Referer": "https://finance.sina.com.cn"})
```

## 响应格式

```
var hq_str_sh600519="贵州茅台,1423.50,1425.00,1430.50,1420.00,1435.00,1423.00,1424.00,1234567,987654321,100,1423.00,200,1424.00,...,2026-05-19,15:00:00,00"
```

字段索引（0-based）：
| 索引 | 字段 |
|------|------|
| 0 | 股票名称 |
| 1 | 开盘价 |
| 2 | 昨收价 |
| 3 | 当前价 |
| 4 | 最高价 |
| 5 | 最低价 |
| 6-7 | 买一/卖一价 |
| 8 | 成交量（手） |
| 9 | 成交额 |
| 10-29 | 十档买卖盘 |
| 30 | 日期 |
| 31 | 时间 |

计算涨跌幅：`(当前价 - 昨收价) / 昨收价 * 100`

## 设置

响应编码为 GBK，需 `.decode("gbk")`。

## N 个股票合并查询

多个代码用逗号拼接在同一 URL 中：
```
http://hq.sinajs.cn/list=sh600519,sz000001,sz300750
```
